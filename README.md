# ☕ DevClub Café

> Um site acolhedor e inspirador feito para desenvolvedores que não vivem sem um bom café de qualidade enquanto programam.

---

## Sobre o Projeto

O **DevClub Café** é uma página web institucional projetada para apresentar uma cafeteria temática voltada para a comunidade de tecnologia. O ambiente une o amor pelo desenvolvimento de software com a paixão por grãos selecionados, oferecendo o espaço perfeito para networking, foco e criatividade.

### 📍 Funcionalidades da Página
- Home:** Apresentação principal com foco na comunidade tech ("Code com o melhor café da região").
- Sobre Nós:** Detalhes sobre o que torna o nosso café especial e focado em produtividade.
- Menu:** Cardápio com os cafés disponíveis e preços promocionais.
- Avaliações:** Espaço para o feedback dos clientes (Devs).
- Endereço:** Localização para os clientes nos encontrarem.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as tecnologias fundamentais da web:

- HTML5:** Estruturação semântica da página.
- CSS3:** Estilização, design responsivo e identidade visual temática.
